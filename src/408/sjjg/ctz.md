## 1.1 数据结构的基本概念

3. 以下算法中最后一句的执行次数为
```c++
int m=0,i,j;
for(int i=1;i<=n;i++)
 for(j=1;j<=2*i;j++)
    m++;
```


错选：D. n^2^

正选：A. n(n+1)

哥们你猪鼻吧，等差数列求和忘得一干二净了

内层循环每次都执行了2i次，而外层循环为n次

即，从2i(i=1)加到2i(i=n)，那么就是...

首项为2，公差为2，末项为2n的等差数列求和

`(2+2n)n/2`=`n(n+1)`

4. 下列函数的时间复杂度是()
```c++
int func(int n){
    int i=0 , sum=0;
    while(sum<n) sum += ++i;
    return i;
}d
```

错选： A. O(logn)

正选： B. O(n^1/2^)

还是等差数列，求得sum=(1+i)*i/2

那么，将i替换为循环次数t(执行了t次)，就有

$$\frac{(1 + t) \cdot t}{2} < n$$



所以时间复杂度为O(n^1/2^)

5. **下列程序段的时间复杂度是(重点)**
 ```c++
 int sum=0
 for(int i=1;i<n;i*=2)
    for(int j=0;j<i;i++)
        sum++;
 
 ```
 错选：C. O(nlogn)

 正选：B. O(n)

别被坑了，看起来是内外双重嵌套循环，实际上内循环可以直接数出来次数为i

那么，可以将这个一样转变为一个等比数列的求和问题

即：1+2+4+...+2^t^

其中，t为内层循环的执行次数，有

$$2^t<n\leq2^{t+1}$$

那么就是...

$$T_{(总执行次数)}=\frac{1 - 2^t}{1-2}=2^{t+1}-1$$

所以有

$$n-1 \leq T 且 T<2n-1$$

## 2.2 线性表的顺序表示

1. Q3 线性表的顺序存储结构是一种()

错选：B. 顺序存取的存储结构

正解：A. 随机存取的存储结构

存取是指读写方式，顺序表支持随机访问，也就是随机存取了！

2. Q9 设线性表中有n个元素，严格来说，以下操作中，()在顺序表上的实现要比在链表上实现的效率高

I.输出第i个元素的值
II. 交换第3个元素与第4个元素的值
III. 顺序输出这n个元素的值

错选：A. I
正选：C. I、II

注意III实际上是同等效率。

II实际上在顺序表上只需要3次交换操作，但是链表效率较低

## 2.3 线性表的链式表示

1. Q3 链式存储设计时，结点内的存储单元地址()
   
错解：C. 不一定连续

正解：A. 一定连续

注意是结点内存储的单元地址，不是结点的地址，对于链表来说，结点内存储的单元地址一定是连续的(不然不叫链表了吧)


2. Q7 给定有n个元素的一维数组，建立一个有序单链表的最低时间复杂度是()

错解：B. O(n)

正解：D. O(nlog~2~n)

对数组先排序再插入需要花费O(log~2~n)·O(n)的时间，不能限定算法输入，因此不能假设原数组已经有序

3. Q12 在线性表a~0~, a~1~, ··· , a~100~中，删除元素a~50~需要移动()个元素

错解：B. 50

正解：D. 0或50

注意没说是顺序表还是链表，如果是链表的话，无需移动元素即可完成删除

4. Q21 对于一个带头结点的循环单链表L，判断该表为空表的条件是()

错解：D. 头结点的指针域与L的地址相等

正解：C. 头结点的指针域与L的值相等

当带头结点的循环单链表L为空表时，满足L->next==L，就是比较值，而不是比对地址！

**指针的值是结点的地址，指针的地址没有意义！**

5. Q22 对于一个带头结点的双循环链表L，判断该表为空表的条件是：()

错解：C. L->prior\==NULL&&L->next\==L

正解：D. L->prior\==L&&L->next\==L

注意审题，这不是双向链表，这是双循环链表，不存在空指针，你个猪鼻

6. Q23 一个链表最常用的操作是在末尾插入结点和删除结点，则选用()最节省时间

错解：C. 带尾指针的单循环链表

正解：A. 带头结点的双循环链表

审题啊猪鼻，人家问的是删除最后的结点，带尾指针的单循环链表删除尾指针所在的结点还要兜一大圈

7. Q25 设有两个长度为n的循环单链表，若要求两个循环单链表的头尾相接时间复杂度为O(1)，则对应两个循环单链表各设置一个指针，分别指向()

8. Q30 分配较大空间，插入和删除元素不需要移动元素的线性表，其存储结构为()
